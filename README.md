
## 功能特性

- ✅ 支持用户名/手机号/邮箱登录
- ✅ 密码显示/隐藏切换
- ✅ 验证码防暴力破解
- ✅ 记住密码功能（7天免登录）
- ✅ 登录失败次数限制和账号锁定
- ✅ 响应式设计（支持桌面端、平板端、移动端）
- ✅ 完整的表单验证和错误提示

## 技术栈

- CSS3
- JavaScript (原生)
